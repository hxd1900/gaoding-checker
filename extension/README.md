# 稿定模板编辑器图层检查 v1.45

**当前版本**：v1.45

## 📋 功能简述

**稿定模板编辑器图层检查** 是一款专为设计师和审核员打造的 Chrome 辅助插件，在原有图层统计基础上，新增了深度视觉审核与智能交互功能，核心能力如下：

1.  **📊 智能统计**：
    *   **精准计数**：实时统计有效图层数量，自动过滤画板、成组及全屏背景干扰。
    *   **自动展开**：页面加载完成后，自动触发并展开左侧原生图层面板，方便人工比对。

2.  **🛡️ 深度视觉审核 (OCR)**：
    *   **文字转曲检测**：集成高精度 OCR 引擎，自动扫描所有图片图层，检测是否包含未转曲的可编辑文字（支持透明背景与大图识别）。
    *   **智能白名单**：支持自定义文字白名单（如“稿定”、“样张”），自动放行合规的装饰性文字，减少误报。

3.  **🕹️ 灵活交互**：
    *   **自由拖拽**：审核面板支持全屏拖拽移动，不再遮挡画布核心区域。
    *   **图钉模式**：新增“固定/取消固定”功能，支持面板常驻显示，或鼠标移开后自动收起为迷你胶囊。

4.  **⚡️ 极速反馈**：
    *   **状态直观**：通过 ✅/❌ 图标即时反馈审核结果，并在面板中直接展示 OCR 识别到的可疑文字内容。

---
*适用于 Chrome 浏览器，安装即用。*
